# DATA_ANALYSIS
Retail Sales Analytics project using Python, Pandas, PostgreSQL to perform data cleaning, exploratory data analysis, SQL-based business insights, and interactive visualizations on a Superstore sales dataset.

# 🛍️ Retail Sales Analytics Project

## 📌 Project Overview

This project demonstrates an end-to-end Retail Sales Analytics workflow using the Superstore Sales Dataset. The objective was to analyze sales data, extract meaningful business insights, and present the findings through SQL queries and visualizations.

The project was completed as part of a Data Analytics Internship and covers essential analytics skills including data cleaning, exploratory data analysis (EDA), SQL querying, and data visualization.

---

## 🎯 Objectives

- Clean and prepare raw sales data for analysis.
- Analyze sales performance using Python and Pandas.
- Answer business questions using SQL queries.
- Create clear and informative visualizations.
- Generate actionable business insights from the dataset.

---

# 📋 Tasks Completed

## ✅ Task 1 – Data Cleaning

Performed data quality checks using Pandas.

Steps completed:

- Loaded the dataset
- Checked data types
- Checked missing values
- Checked duplicate records
- Verified dataset consistency
- Exported cleaned dataset

### Result

- Missing Values: **0**
- Duplicate Records: **0**

The dataset required no additional cleaning as it was already well-structured.

---

## ✅ Task 2 – Sales Data Analysis

Performed exploratory data analysis (EDA) using Pandas.

Key analyses included:

- Total Sales
- Average Order Value
- Total Profit
- Sales by Category
- Sales by Region
- Monthly Sales Trend
- Top Products
- Top Customers
- Quantity Sold
- Average Discount

---

## ✅ Task 4 – SQL Queries for Business Insights

Used PostgreSQL to answer business questions using SQL.

SQL concepts covered:

- Aggregate Functions
- GROUP BY
- ORDER BY
- LIMIT
- Business Reporting Queries

Example business questions:

- What are the total sales?
- Which region generated the highest revenue?
- Which category is the most profitable?
- Who are the top customers?
- Which products generate the highest profit?

---

## ✅ Task 6 – Data Visualization

Created visualizations using Matplotlib.

Charts included:

- Sales by Category
- Sales by Region
- Monthly Sales Trend
- Top 10 Products by Profit

These visualizations help identify sales trends and support data-driven decision-making.

---

# 📊 Key Insights

- Technology generated the highest sales revenue.
- Certain regions consistently outperformed others in sales.
- A small group of customers contributed significantly to overall revenue.
- Monthly sales trends highlighted seasonal fluctuations.
- High-profit products can be prioritized for future sales strategies.

# ▶️ How to Run the Project

1. Clone the repository.

```
git clone <repository-url>
```

2. Install the required Python libraries.

```
pip install -r requirements.txt
```

3. Open the Jupyter notebooks.

4. Execute the notebooks in order:

- Task1_Data_Cleaning.ipynb
- Task2_Sales_Analysis.ipynb
- Task6_Visualization.ipynb

5. Execute the SQL queries in PostgreSQL using pgAdmin.
